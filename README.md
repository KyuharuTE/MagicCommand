# 📖 使用说明
- 安装 1.20.4 Fabric + Fabric API
- 将编译好的 Jar 文件拖入 mods 文件夹
- 启动游戏
- 在游戏中按下 `M` 打开界面

# 🔨 构建环境
- Java 17
  - okhttp 3
  - org.json
- Minecraft 1.20.4
  - yarn_mappings 1.20.4+build.3
  - loader_version 0.16.10
  - fabric_version 0.97.2+1.20.4

# ✅ 实现功能
- 自动生成命令
- 与 Ai 对话
- 生成多条命令并运行
